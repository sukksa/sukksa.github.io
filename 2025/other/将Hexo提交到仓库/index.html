е°Ҷ Hexo жҸҗдәӨеҲ°д»“еә“

иҝҷжҳҜдёҖдёӘ Hexo еҸҢд»“еә“з®ЎзҗҶй—®йўҳпјҲдёҖдёӘжҳҜз”ЁдәҺеӯҳж”ҫйқҷжҖҒзҪ‘йЎөзҡ„ xxx.github.ioпјҢдёҖдёӘжҳҜз”ЁдәҺеӯҳж”ҫжәҗд»Јз Ғзҡ„ hexo-codeпјүгҖӮ

жҸҗдәӨдёҚдёҠеҺ»жҳҜеӣ дёәGit й…ҚзҪ®еҶІзӘҒжҲ–иҖ…**.gitignore и®ҫзҪ®дёҚеҪ“**йҖ жҲҗзҡ„гҖӮ

еҲӣе»ә .gitignore ж–Үд»¶

Hexo йғЁзҪІе’Ңжәҗз ҒеӨҮд»ҪжңҖеёёи§Ғзҡ„й—®йўҳе°ұжҳҜжҠҠдёҚиҜҘдј зҡ„ж–Үд»¶дј дәҶпјҢжҲ–иҖ…иҜҘдј зҡ„ж–Үд»¶иў«еҝҪз•ҘдәҶгҖӮ

жү“ејҖжҲ–еңЁж №зӣ®еҪ•дёӢж–°е»ә .gitignore ж–Үд»¶пјҢзЎ®дҝқйҮҢйқўеҝ…йЎ»еҢ…еҗ«д»ҘдёӢеҶ…е®№гҖӮ

  • public/ е’Ң .deploy_git/пјҡиҝҷдёӨдёӘж–Үд»¶еӨ№жҳҜ hexo d з”ҹжҲҗзҡ„йқҷжҖҒж–Үд»¶е’ҢйғЁзҪІз”Ёзҡ„ git д»“еә“пјҢз»қеҜ№дёҚиғҪжҸҗдәӨеҲ°дҪ зҡ„жәҗз Ғд»“еә“ hexo-code дёӯпјҢеҗҰеҲҷдјҡйҖ жҲҗдёҘйҮҚзҡ„еөҢеҘ—еҶІзӘҒгҖӮ
  • **node_modules/**пјҡиҝҷжҳҜдҫқиө–еҢ…пјҢдҪ“з§ҜеҫҲеӨ§дё”дёҚйңҖиҰҒеӨҮд»ҪпјҢеҸӘйңҖеӨҮд»Ҫ package.json еҚіеҸҜгҖӮ

еӨ„зҗҶдё»йўҳпјҲThemesпјүж–Үд»¶еӨ№зҡ„ Git еҶІзӘҒ

иҝҷжҳҜжңҖе®№жҳ“еҜјиҮҙвҖңж— жі•жҸҗдәӨвҖқжҲ–вҖңжҸҗдәӨеҗҺж–Үд»¶еӨ№жҳҜз©әзҡ„вҖқзҡ„еҺҹеӣ гҖӮ

еҰӮжһңдҪ дёӢиҪҪзҡ„дё»йўҳжҳҜйҖҡиҝҮ git clone дёӢиҪҪеҲ° themes/ зӣ®еҪ•дёӢзҡ„пјҢйӮЈд№ҲиҜҘдё»йўҳж–Үд»¶еӨ№йҮҢдјҡжңүдёҖдёӘйҡҗи—Ҹзҡ„ .git ж–Үд»¶еӨ№гҖӮ

и§ЈеҶіж–№жі•пјҡиҝӣе…Ҙ themes/дҪ зҡ„дё»йўҳеҗҚ/ зӣ®еҪ•гҖӮеҲ жҺүдё»йўҳдёӢзҡ„ .git ж–Үд»¶еӨ№пјҢдёҚиҰҒеҲ жҺүж №зӣ®еҪ•зҡ„пјүгҖӮ

иҝҷж ·еҒҡжҳҜдёәдәҶи®©дё»д»“еә“жҠҠдё»йўҳж–Үд»¶еҪ“дҪңжҷ®йҖҡж–Үд»¶жқҘз®ЎзҗҶпјҢиҖҢдёҚжҳҜеҪ“дҪңдёҖдёӘеӯҗжЁЎеқ—пјҲSubmoduleпјүгҖӮ

з”ҹжҲҗ SSH е…¬й’ҘиҝһжҺҘеҲ° GitHub

жЈҖжҹҘе№¶з”ҹжҲҗ SSH еҜҶй’Ҙ

иҜ·еңЁдҪ зҡ„ Git Bash (жҲ–иҖ…з»Ҳз«Ҝ) дёӯдҫқж¬Ўжү§иЎҢд»ҘдёӢе‘Ҫд»Өпјҡ

  1. з”ҹжҲҗж–°еҜҶй’ҘпјҲиҜ·зӣҙжҺҘеӨҚеҲ¶дёӢж–№е‘Ҫд»ӨпјҢе°Ҷе…¶дёӯзҡ„ дҪ зҡ„йӮ®з®ұ жҚўжҲҗдҪ  GitHub жіЁеҶҢзҡ„йӮ®з®ұпјүпјҡ

  2. дёҖи·ҜеӣһиҪҰпјҡ

    • зі»з»ҹдјҡй—®дҪ дҝқеӯҳи·Ҝеҫ„ -> жҢүеӣһиҪҰпјҲй»ҳи®ӨеҚіеҸҜпјүгҖӮ
    • зі»з»ҹдјҡй—®дҪ жҳҜеҗҰиҰҶзӣ– -> еҰӮжһңжҸҗзӨә overwriteпјҢиҫ“е…Ҙ y з„¶еҗҺеӣһиҪҰпјӣеҰӮжһңжІЎжңүжҸҗзӨәпјҢзӣҙжҺҘдёӢдёҖжӯҘгҖӮ
    • зі»з»ҹдјҡй—®дҪ  PassphraseпјҲеҜҶз Ғпјү -> жҢүеӣһиҪҰпјҲз•ҷз©әпјҢдёҚиҰҒи®ҫеҜҶз ҒпјҢеҗҰеҲҷд»ҘеҗҺжҜҸж¬ЎйғЁзҪІйғҪиҰҒиҫ“еҜҶз ҒпјүгҖӮ
    • еҶҚж¬ЎзЎ®и®ӨеҜҶз Ғ -> жҢүеӣһиҪҰгҖӮ

    зӣҙеҲ°зңӢи§ҒдёҖдёӘзҹ©еҪўзҡ„еӯ—з¬Ұз”»пјҲRandomart imageпјүпјҢиҜҙжҳҺз”ҹжҲҗжҲҗеҠҹдәҶгҖӮ

иҺ·еҸ–е…¬й’ҘеҶ…е®№

з»§з»ӯеңЁз»Ҳз«Ҝиҫ“е…Ҙд»ҘдёӢе‘Ҫд»ӨпјҢжҹҘзңӢдҪ зҡ„е…¬й’ҘеҶ…е®№пјҡ

з»Ҳз«ҜдјҡжҳҫзӨәд»Ҙ ssh-ed25519 ејҖеӨҙзҡ„дёҖй•ҝдёІеӯ—з¬ҰгҖӮиҜ·жҠҠиҝҷдёҖж•ҙж®өеҶ…е®№пјҲеҢ…жӢ¬ејҖеӨҙзҡ„ ssh-ed25519 е’Ңз»“е°ҫзҡ„йӮ®з®ұпјүе…ЁйғЁеӨҚеҲ¶дёӢжқҘгҖӮ

ж·»еҠ еҲ° GitHub

  1. жү“ејҖжөҸи§ҲеҷЁпјҢзҷ»еҪ•дҪ зҡ„ GitHub иҙҰеҸ·гҖӮ
  2. зӮ№еҮ»еҸідёҠи§’еӨҙеғҸ -> Settings (и®ҫзҪ®)гҖӮ
  3. еңЁе·Ұдҫ§иҸңеҚ•ж ҸжүҫеҲ° SSH and GPG keysгҖӮ
  4. зӮ№еҮ»з»ҝиүІзҡ„ New SSH key жҢүй’®гҖӮ
  5. TitleпјҡйҡҸдҫҝеЎ«пјҲдҫӢеҰӮпјҡMy LaptopпјүгҖӮ
  6. KeyпјҡзІҳиҙҙдҪ еңЁз¬¬дәҢжӯҘеӨҚеҲ¶зҡ„йӮЈдёҖй•ҝдёІеҶ…е®№гҖӮ
  7. зӮ№еҮ» Add SSH keyгҖӮ

йӘҢиҜҒ

еӣһеҲ°дҪ зҡ„ Git Bash з»Ҳз«ҜпјҢиҫ“е…Ҙд»ҘдёӢе‘Ҫд»ӨжөӢиҜ•иҝһжҺҘпјҡ

  • еҰӮжһңе®ғй—® Are you sure...пјҢеҶҚж¬Ўиҫ“е…Ҙ yes е№¶еӣһиҪҰгҖӮ

  • жҲҗеҠҹзҡ„ж Үеҝ—пјҡдҪ дјҡзңӢеҲ°иҝҷж ·дёҖеҸҘиҜқпјҡ

    Hi <дҪ зҡ„з”ЁжҲ·еҗҚ>! You've successfully authenticated, but GitHub does not provide shell access.

йҮҚж–°йғЁзҪІ Hexo

иҜ·еӣһеҲ° Hexo зӣ®еҪ•пјҢеҶҚж¬Ўе°қиҜ•пјҡ

е°ұиғҪйЎәеҲ©дёҠдј дәҶгҖӮ

жё…йҷӨеҮӯиҜҒ

жё…йҷӨжң¬ең°зҡ„ GitHub иә«д»ҪеҮӯиҜҒпјҲжңҖе…ій”®пјү

дҪ зҡ„з”өи„‘дёҠжңүдёӨжҠҠвҖңй’ҘеҢҷвҖқеҸҜд»Ҙи®ҝй—®дҪ зҡ„ GitHub иҙҰжҲ·пјҡSSH з§Ғй’Ҙе’ҢGit еҮӯиҜҒз®ЎзҗҶеҷЁгҖӮ

  1. еҲ йҷӨ SSH з§Ғй’Ҙж–Үд»¶

вҖӢ дҪ д№ӢеүҚдёәдәҶйғЁзҪІ Hexo иҖҢеҲӣе»әзҡ„ SSH еҜҶй’ҘеҜ№пјҢе…¶дёӯз§Ғй’Ҙж–Үд»¶ (id_ed25519) еӯҳж”ҫеңЁдҪ зҡ„з”ЁжҲ·дё»зӣ®еҪ•дёӢгҖӮ

  • и·Ҝеҫ„пјҡ C:\Users\дҪ зҡ„з”ЁжҲ·еҗҚ\.ssh\ ж–Үд»¶еӨ№
  • йңҖиҰҒеҲ йҷӨзҡ„ж–Үд»¶пјҡ
    • id_ed25519 (иҝҷжҳҜдҪ зҡ„з§Ғй’ҘпјҢеҠЎеҝ…еҲ йҷӨпјҒ)
    • id_ed25519.pub (иҝҷжҳҜе…¬й’ҘпјҢеҲ йҷӨе®ғд№ҹж— еҰЁпјҢдҪҶз§Ғй’ҘжҳҜе…ій”®)
    • known_hosts (и®°еҪ•дәҶиҝһжҺҘиҝҮзҡ„жңҚеҠЎеҷЁжҢҮзә№пјҢеҲ йҷӨеҸҜжё…зҗҶиҝһжҺҘи®°еҪ•)
  1. жё…йҷӨ Git еҮӯиҜҒзј“еӯҳ

вҖӢ еҰӮжһңдҪ дҪҝз”Ё HTTPS ж–№ејҸе…ӢйҡҶд»“еә“пјҢжҲ–иҖ…еңЁйғЁзҪІж—¶иҫ“е…ҘиҝҮ GitHub зҡ„з”ЁжҲ·еҗҚе’ҢеҜҶз Ғ/дёӘдәәи®ҝй—®д»ӨзүҢ (PAT)пјҢзі»з»ҹеҸҜиғҪдјҡе°Ҷе…¶зј“еӯҳеңЁ Windows зҡ„еҮӯиҜҒз®ЎзҗҶеҷЁдёӯгҖӮ

  1. жү“ејҖ Windows жҗңзҙўж ҸпјҢжҗңзҙў вҖңеҮӯиҜҒз®ЎзҗҶеҷЁвҖқ (Credential Manager)гҖӮ
  2. йҖүжӢ© вҖңWindows еҮӯиҜҒвҖқ (Windows Credentials)гҖӮ
  3. еңЁвҖңжҷ®йҖҡеҮӯиҜҒвҖқ (Generic Credentials) йғЁеҲҶпјҢжҹҘжүҫд»»дҪ•д»Ҙ git:https://github.com жҲ– github.com ејҖеӨҙзҡ„жқЎзӣ®гҖӮ
  4. жүҫеҲ°еҗҺзӮ№еҮ»е№¶йҖүжӢ© вҖңз§»йҷӨвҖқ (Remove) жҲ– вҖңеҲ йҷӨвҖқ (Delete)гҖӮ

д»Һ GitHub иҙҰжҲ·дёӯеҲ йҷӨе…¬й’ҘпјҲиҝңзЁӢж“ҚдҪңпјү

еҚідҪҝжң¬ең°зҡ„з§Ғй’Ҙж–Үд»¶иў«еҲ йҷӨпјҢдёәдәҶйҳІжӯўд»»дҪ•дәәйҖҡиҝҮе…¶д»–ж–№ејҸжҒўеӨҚиҜҘж–Үд»¶пјҢдҪ иҝҳйңҖиҰҒд»Һ GitHub иҙҰжҲ·дёӯз§»йҷӨеҜ№еә”зҡ„е…¬й’ҘгҖӮиҝҷж ·пјҢеҚідҪҝз§Ғй’Ҙжі„йңІпјҢе®ғд№ҹж— жі•з”ЁдәҺи®ҝй—®дҪ зҡ„ GitHub иҙҰжҲ·гҖӮ

  1. жү“ејҖжөҸи§ҲеҷЁпјҢзҷ»еҪ•дҪ зҡ„ GitHub иҙҰжҲ·гҖӮ
  2. зӮ№еҮ»еҸідёҠи§’еӨҙеғҸпјҢиҝӣе…Ҙ Settings (и®ҫзҪ®)гҖӮ
  3. еңЁе·Ұдҫ§иҸңеҚ•ж ҸжүҫеҲ° SSH and GPG keysгҖӮ
  4. жүҫеҲ°дҪ д№ӢеүҚж·»еҠ зҡ„гҖҒз”ЁдәҺиҝҷеҸ°з”өи„‘зҡ„ SSH Key зЎ®и®ӨеҲ йҷӨ
