еҺҹз”ҹAjax

Ajax е…Ёз§° Asynchronous JavaScript and XML(ејӮжӯҘзҡ„ JavaScript е’Ң XML)пјҢж—ўжҳҜдёҖдёӘеҜ№иұЎпјҢд№ҹжҳҜдёҖз§Қж–№жі•жЁЎејҸ

AJAX жҳҜдёҖз§Қз”ЁдәҺеҲӣе»әеҝ«йҖҹеҠЁжҖҒзҪ‘йЎөзҡ„жҠҖжңҜпјҢйҖҡиҝҮеңЁеҗҺеҸ°дёҺжңҚеҠЎеҷЁиҝӣиЎҢе°‘йҮҸж•°жҚ®дәӨжҚў,Ajax еҸҜд»ҘдҪҝзҪ‘йЎөе®һзҺ°ејӮжӯҘжӣҙж–°гҖӮиҝҷж„Ҹе‘ізқҖеҸҜд»ҘеңЁдёҚйҮҚж–°еҠ иҪҪж•ҙдёӘзҪ‘йЎөзҡ„жғ…еҶөдёӢпјҢеҜ№зҪ‘йЎөзҡ„жҹҗйғЁеҲҶиҝӣиЎҢжӣҙж–°пјҢдј з»ҹзҡ„дёҚдҪҝз”Ё Ajax зҡ„зҪ‘йЎөеҰӮжһңйңҖиҰҒжӣҙж–°еҶ…е®№,еҝ…йңҖйҮҚиҪҪж•ҙдёӘзҪ‘йЎө

з”Ёжі•

еҲӣе»ә Ajax еҜ№иұЎ

йҖҡиҝҮ XMLHttpRequest()зұ»еҲӣе»әдёҖдёӘ Ajax еҜ№иұЎ(жӯӨж–№жі•дёҚе…је®№ IE6 д»ҘдёӢжөҸи§ҲеҷЁ)

    var Ajax=new XMLHttpRequest()

еңЁ IE6 д»ҘдёӢжөҸи§ҲеҷЁйҖҡиҝҮ ActiveXObject("Microsoft.XMLHTTP")еҲӣе»әдёҖдёӘ Ajax еҜ№иұЎ

    var Ajax=new ActiveXObject("Microsoft.XMLHTTP")

еҲқе§ӢеҢ–

йҖҡиҝҮ open() ж–№жі•пјҢеҸҜд»Ҙи®ҫзҪ®и·ҹеҗҺеҸ°дәӨдә’зҡ„дёҖдәӣиЎҢдёә,иҜҘж–№жі•жңүдёүдёӘеҸӮж•°
1гҖҒmethodпјӣ2гҖҒurlпјӣ3гҖҒboolean

method иҜ·жұӮж–№ејҸ,дёҖдёӘеҢ…еҗ«е…·дҪ“иҜ·жұӮж–№ејҸзҡ„еӯ—з¬ҰдёІ

  • дё»иҰҒиҜ·жұӮж–№ејҸпјҡ вҖӢ
    • POST,е»әи®®еңЁж·»еҠ ж•°жҚ®ж—¶дҪҝз”Ё
    • GET,е»әи®®еңЁжҹҘиҜўж•°жҚ®ж—¶дҪҝз”Ё вҖӢ
    • PUT,е»әи®®еңЁдҝ®ж”№ж•°жҚ®ж—¶дҪҝз”Ё вҖӢ
    • DELETE,е»әи®®еңЁеҲ йҷӨж•°жҚ®ж—¶дҪҝз”Ё

url,иҰҒиҜ·жұӮж•°жҚ®зҡ„е…·дҪ“ең°еқҖ

bool,иҝӣиЎҢеҗҢжӯҘжҲ–ејӮжӯҘдј иҫ“,true дёәејӮжӯҘ,false дёәеҗҢжӯҘ

    Ajax.open("GET", "http://localhost:8000", true)

и®ҫзҪ®йңҖжұӮ

    // еҰӮжһңжғіз”ЁGETзҡ„ж—¶еҖҷзӣҙжҺҘж·»еҠ йңҖжұӮ,еҸҜд»ҘеңЁurlеҗҺйқўеҠ дёҠ?еҶҚеҶҷе…·дҪ“зҡ„йңҖжұӮ,еӨҡдёӘйңҖжұӮйҖҡиҝҮ&з¬ҰеҸ·иҝһжҺҘ
    var Ajax = new XMLHttpRequest()
    Ajax.open("GET","https://www.baidu.com?user=zhangsan&pwd=123456",true)

    // иЎЁеҚ•ж јејҸиҜ·жұӮеӨҙдҝЎжҒҜзҡ„еҶҷжі•
    
    Ajax.setRequestHeader("Content-Type","application/x-www-form-urlencoded"); //urlж јејҸ

    Ajax.send("user=zhangsan&pwd=123456")

    // еҰӮжһңжғіз”ЁGETзҡ„ж—¶еҖҷзӣҙжҺҘж·»еҠ йңҖжұӮ,еҸҜд»ҘеңЁurlеҗҺйқўеҠ дёҠ?еҶҚеҶҷе…·дҪ“зҡ„йңҖжұӮ,еӨҡдёӘйңҖжұӮйҖҡиҝҮ&з¬ҰеҸ·иҝһжҺҘ
    Ajax.open("GET","https://www.baidu.com?user=zhangsan&pwd=123456",true)

    // иЎЁеҚ•ж јејҸиҜ·жұӮеӨҙдҝЎжҒҜзҡ„еҶҷжі•

GET дёҺ POST еҢәеҲ«
дёҺ POST зӣёжҜ”пјҢGET жӣҙз®ҖеҚ•д№ҹжӣҙеҝ«пјҢе№¶дё”еңЁеӨ§йғЁеҲҶжғ…еҶөдёӢйғҪиғҪз”ЁгҖӮ
з„¶иҖҢпјҢеңЁд»ҘдёӢжғ…еҶөдёӯпјҢиҜ·дҪҝз”Ё POST иҜ·жұӮпјҡ

  • ж— жі•дҪҝз”Ёзј“еӯҳж–Үд»¶пјҲжӣҙж–°жңҚеҠЎеҷЁдёҠзҡ„ж–Үд»¶жҲ–ж•°жҚ®еә“пјү
  • еҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒеӨ§йҮҸж•°жҚ®пјҲPOST жІЎжңүж•°жҚ®йҮҸйҷҗеҲ¶пјү
  • еҸ‘йҖҒеҢ…еҗ«жңӘзҹҘеӯ—з¬Ұзҡ„з”ЁжҲ·иҫ“е…Ҙж—¶пјҢPOST жҜ” GET жӣҙзЁіе®ҡд№ҹжӣҙеҸҜйқ 

еҸ‘йҖҒиҜ·жұӮ

йҖҡиҝҮ send()ж–№жі•жӯЈејҸеҸ‘йҖҒиҜ·жұӮ

    Ajax.send()

жіЁ: еҰӮжһңиҜ·жұӮж–№ејҸжҳҜ POST ж–№ејҸ,ж·»еҠ зҡ„йңҖжұӮдҪңдёәеҸӮж•°еҶҷе…Ҙ send()ж–№жі•дёӯ

    Ajax.send('a=100&b=200&c=300')

иҝ”еӣһиҜ·жұӮз»“жһң

жҜҸеҪ“ readyState ж”№еҸҳж—¶пјҢе°ұдјҡи§ҰеҸ‘ onreadystatechange дәӢд»¶пјҢеӣ жӯӨеҸҜд»Ҙз»ҷе…ғзҙ з»‘е®ҡonreadystatechangeдәӢд»¶пјҢжқҘжҺҘеҸ—е“Қеә”зҡ„дәӢд»¶
XMLHttpRequest еҜ№иұЎжңүдёүдёӘйқһеёёйҮҚиҰҒзҡ„еұһжҖ§пјҡonreadystatechangeгҖҒreadyStateгҖҒstatus

readyStateеұһжҖ§пјҡ

йҖҡиҝҮ status(http зҡ„зҠ¶жҖҒз Ғ)еұһжҖ§зҡ„еҖјеҲӨж–ӯиҜ·жұӮеҗҺзҡ„з»“жһңд»ҺиҖҢеҫ—еҲ°иҜ·жұӮдҝЎжҒҜ,дёҖиҲ¬иҜҘеҖјдёә 200 еҲ° 300 зӣҙжҺҘе’ҢзӯүдәҺ 304 ж—¶д»ЈиЎЁиҜ·жұӮжҲҗеҠҹ,еҖјдёә 404 д»ЈиЎЁиҜ·жұӮеӨұиҙҘ

response еұһжҖ§дҝқз•ҷзқҖеҗҺеҸ°иҝ”еӣһеҲ°еүҚз«Ҝзҡ„ж•°жҚ®

    Ajax.onreadystatechange = function () {
        if (Ajax.readyState === 4) {
            //зЎ®е®ҡhttpзҡ„зҠ¶жҖҒз Ғ
            if ((Ajax.status >= 200 && Ajax.status < 300) || Ajax.status === 304) {
                console.log(Ajax.response);
            }
        }
    }

XmlHttpRequest еҜ№иұЎзҡ„дё»иҰҒж–№жі•дёҺеұһжҖ§

XmlHttpRequest еҜ№иұЎзҡ„дё»иҰҒж–№жі•

  • void open(String method,String url,Boolen async)з”ЁдәҺеҲӣе»әиҜ·жұӮ
    еҸӮж•°пјҡ

    • methodпјҡ иҜ·жұӮж–№ејҸпјҲеӯ—з¬ҰдёІзұ»еһӢпјүпјҢеҰӮпјҡPOSTгҖҒGETгҖҒDELETEвҖҰ
    • urlпјҡ иҰҒиҜ·жұӮзҡ„ең°еқҖпјҲеӯ—з¬ҰдёІзұ»еһӢпјү
    • asyncпјҡ жҳҜеҗҰејӮжӯҘпјҲеёғе°”зұ»еһӢпјү

  • void send(String body)з”ЁдәҺеҸ‘йҖҒиҜ·жұӮ
    еҸӮж•°пјҡ

    • bodyпјҡ иҰҒеҸ‘йҖҒзҡ„ж•°жҚ®пјҲеӯ—з¬ҰдёІзұ»еһӢпјү

  • void setRequestHeader(String header,String value) з”ЁдәҺи®ҫзҪ®иҜ·жұӮеӨҙ
    еҸӮж•°пјҡ

    • headerпјҡ иҜ·жұӮеӨҙзҡ„ keyпјҲеӯ—з¬ҰдёІзұ»еһӢпјү
    • vlaueпјҡ иҜ·жұӮеӨҙзҡ„ valueпјҲеӯ—з¬ҰдёІзұ»еһӢпјү

  • String getAllResponseHeaders() иҺ·еҸ–жүҖжңүе“Қеә”еӨҙ
    иҝ”еӣһеҖјпјҡ

    • е“Қеә”еӨҙж•°жҚ®пјҲеӯ—з¬ҰдёІзұ»еһӢпјү

  • String getResponseHeader(String header) иҺ·еҸ–е“Қеә”еӨҙдёӯжҢҮе®ҡ header зҡ„еҖј
    еҸӮж•°пјҡ

    • headerпјҡ е“Қеә”еӨҙзҡ„ keyпјҲеӯ—з¬ҰдёІзұ»еһӢпјү
      иҝ”еӣһеҖјпјҡ
    • е“Қеә”еӨҙдёӯжҢҮе®ҡзҡ„ header еҜ№еә”зҡ„еҖј

  • void abort() з»ҲжӯўиҜ·жұӮ

XmlHttpRequest еҜ№иұЎзҡ„дё»иҰҒеұһжҖ§

  • Number readyState зҠ¶жҖҒеҖјпјҲж•ҙж•°пјү
    иҜҰз»Ҷпјҡ

    • 0-жңӘеҲқе§ӢеҢ–пјҢе°ҡжңӘи°ғз”Ё open()ж–№жі•пјӣ
    • 1-еҗҜеҠЁпјҢи°ғз”ЁдәҶ open()ж–№жі•пјҢжңӘи°ғз”Ё send()ж–№жі•пјӣ
    • 2-еҸ‘йҖҒпјҢе·Із»Ҹи°ғз”ЁдәҶ send()ж–№жі•пјҢжңӘжҺҘж”¶еҲ°е“Қеә”пјӣ
    • 3-жҺҘж”¶пјҢе·Із»ҸжҺҘж”¶еҲ°йғЁеҲҶе“Қеә”ж•°жҚ®пјӣ
    • 4-е®ҢжҲҗпјҢе·Із»ҸжҺҘж”¶еҲ°е…ЁйғЁе“Қеә”ж•°жҚ®пјӣ

  • Function onreadystatechange еҪ“ readyState зҡ„еҖјж”№еҸҳж—¶иҮӘеҠЁи§ҰеҸ‘жү§иЎҢе…¶еҜ№еә”зҡ„еҮҪж•°пјҲеӣһи°ғеҮҪж•°пјү

  • String responseTextжңҚеҠЎеҷЁиҝ”еӣһзҡ„ж•°жҚ®пјҲеӯ—з¬ҰдёІзұ»еһӢпјү

  • XmlDocument responseXMLжңҚеҠЎеҷЁиҝ”еӣһзҡ„ж•°жҚ®пјҲXml еҜ№иұЎпјү

  • Number states зҠ¶жҖҒз ҒпјҲж•ҙж•°пјүпјҢеҰӮпјҡ200гҖҒ404вҖҰ

  • String statesTextзҠ¶жҖҒж–Үжң¬пјҲеӯ—з¬ҰдёІпјүпјҢеҰӮпјҡOKгҖҒNotFoundвҖҰ

жҖ»з»“

  1. е®ҡд№ү XMLHttpRequest еҜ№иұЎ
  2. еҲқе§ӢеҢ– и®ҫзҪ®иҜ·жұӮж–№жі•е’ҢURL пјҲopen() ж–№жі•пјү
  3. еҸ‘йҖҒиҜ·жұӮ пјҲsend() ж–№жі•пјү
  4. жіЁеҶҢзӣ‘еҗ¬зҡ„еӣһи°ғеҮҪж•°
  5. жӢҝеҲ°иҝ”еӣһеҖјпјҢеҜ№йЎөйқўиҝӣиЎҢж“ҚдҪң
